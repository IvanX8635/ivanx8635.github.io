Freedom Project

This was a year long project where I research aspects of physics and applied it to a website

There were many things that were challenging. I learned a new tool called Jekyll. Jekyll is a tool that is based off of ruby that creates a static site where you can use markdown to help you quickly update the theme of the website or the overall content. At first, it was challenging to learn because I was learning the tool all by myself. From this experience, I learned how to learn on my own. I reached out for help on websites such as stackoverflow and many other sources. The pandemic hits pretty hard, but I persevered!

This project allowed to create blog entries to log my progress on the project. It makes it so that others can look at the steps I took to get to a milestone.
